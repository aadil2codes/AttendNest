# 📚 Smart Attendance Tracker

A simple and smart Progressive Web App (PWA) built to help students track attendance, monitor attendance percentage, and stay on top of attendance requirements with automatic reminders.

## 🚀 Live Demo

🔗 https://attendnest.vercel.app

---

## ✨ Features

### 📖 Subject Management

* Add unlimited subjects
* Delete subjects anytime
* Separate attendance records for each subject

### 📅 Smart Calendar System

* Monthly calendar view
* Navigate between months
* Mark attendance for any date
* Today's date highlighted automatically

### ✅ Attendance Tracking

* Present
* Absent
* No Class / Government Holiday
* Clear attendance status

### 📊 Attendance Analytics

* Total classes attended
* Total classes conducted
* Attendance percentage calculation
* Color-coded attendance indicators

### 🧠 Smart Subject Scheduling

* Regular subjects with multiple class days
* Weekly subjects with a single class day
* Automatic class-day detection
* Timetable-based attendance tracking

### 🔔 Smart Reminders

* Reminder banner after 6 PM
* Only shows on actual class days
* Prevents forgetting attendance marking
* Background notification support through Service Workers

### 📱 Progressive Web App (PWA)

* Installable on mobile and desktop
* App-like experience
* Works independently from browser tabs
* Fast and lightweight

## 🎯 How It Works

1. Add a subject.
2. Select whether it is:

   * Regular Subject
   * Weekly Subject
3. Choose class days.
4. Open the subject calendar.
5. Mark attendance as:

   * Present
   * Absent
   * No Class
6. Track attendance percentage automatically.
7. Receive reminders if attendance is not marked after 6 PM.
